## Quick Start

**For Users**: See [USER_GUIDE.md](USER_GUIDE.md) for detailed usage instructions.

**For Developers**: See [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md) for setup and development instructions.

### Basic Setup

#### 1. Install Dependencies

```bash
# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install Python packages
pip install -r requirements.txt
```

### 2. Configure Environment Variables

Create a `.env` file in the root directory (or use the UI to input API key):

```env
API_KEY=your_api_key_here
API_MODEL=gpt-4o-mini
TRENDING_PROFILES=@profile1,@profile2,@profile3
```

**Note**: You can also input the API key directly in the UI - no `.env` file is required.

### 3. Start the Backend Server

```bash
cd backend
python main.py
```

Or using uvicorn directly:

```bash
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

The API will be available at `http://localhost:8000`

### 4. Open the Frontend

Simply open `frontend/index.html` in your web browser, or serve it using a local server:

```bash
# Using Python
cd frontend
python -m http.server 8080

# Then open http://localhost:8080 in your browser
```

Or use any other static file server of your choice.

## Usage

1. **Enter API Configuration**
   - Input your API key (OpenAI, Anthropic, or Google)
   - Select your preferred API model from the dropdown

2. **Create Content**
   - Enter a topic for your content
   - Select content type (Reel, Carousel, Story, or Post)
   - Click "Generate Content"

3. **Review Generated Content**
   - View generated script, caption, hashtags, and alt text
   - Copy content for use in your posts

4. **Track Checklist**
   - Monitor automated tasks (script, caption, hashtags, alt text)
   - Mark manual tasks as complete when done
   - Reference `manual_tasks.md` for tasks outside the app

## Supported API Providers

- **OpenAI**: gpt-4o, gpt-4o-mini, gpt-4-turbo, gpt-3.5-turbo
- **Anthropic**: claude-3-5-sonnet-20241022, claude-3-opus-20240229
- **Google**: gemini-pro

## Content Creation Workflow

The app automates the following steps:

1. âœ… **Generate Idea** - Based on your topic
2. âœ… **Generate Script/Content** - AI-generated script for Reels or content for Carousels
3. âœ… **Generate Caption** - Catchy captions with emojis

## Checklist System

The app includes a checklist for every content creation session:

- **Automated tasks**: Generated by the app (script, caption, hashtags, alt text)
- **Manual tasks**: External tools or Instagram actions
- **Status tracking**: Pending, Completed, Failed, Manual
- **Manual marking**: Mark any task as "manual" if automation doesn't work

## Customization

### Adding Trending Profiles

Edit `backend/constants.py` or set `TRENDING_PROFILES` environment variable:

```python
TRENDING_PROFILES = "@profile1,@profile2,@profile3"
```

### Modifying Checklist

Edit `CONTENT_CREATION_CHECKLIST` in `backend/constants.py` to add or modify checklist items.

### Custom Prompts

Modify `PROMPT_TEMPLATES` in `backend/constants.py` to customize AI prompts.

## Troubleshooting

### Backend not starting
- Ensure all dependencies are installed: `pip install -r requirements.txt`
- Check if port 8000 is available
- Verify Python version (3.8+)

### API errors
- Verify your API key is correct
- Check API rate limits
- Ensure you have credits/quota for the selected model

### CORS errors
- The backend allows all origins by default for personal use
- If issues persist, check browser console for specific errors

## Notes

- This is a personal use application - no authentication required
- Checklist state is stored in-memory (resets on server restart)
- For production use, consider adding database persistence
- All video/audio creation must be done using external tools (see `manual_tasks.md`)
