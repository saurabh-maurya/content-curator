# Manual Tasks - Outside App Scope

This document lists tasks that are **NOT** part of this automation app and need to be completed manually.

## External Tool Tasks

These tasks require external tools/services that are not integrated in this app:

1. **Create Voiceover (TTS)**
   - Tool: TTSMaker / ElevenLabs Free / Narakeet
   - Action: Convert script to voice MP3
   - Manual step: Upload script to TTS service and download audio

2. **Build Avatar Video**
   - Tool: HeyGen Free / CapCut AI Avatar / Pika Labs
   - Action: Create talking avatar video
   - Manual step: Upload photo, add script, generate video

3. **Edit Video**
   - Tool: CapCut (Desktop/Web)
   - Action: Add captions, emojis, transitions, trending audio
   - Manual step: Import video, edit, export 1080x1920 MP4

4. **Get Stock Footage**
   - Tool: Pexels / Pixabay / Mixkit
   - Action: Download free commercial videos
   - Manual step: Search and download relevant footage

5. **Get Music/SFX**
   - Tool: YouTube Audio Library / Instagram Music
   - Action: Select trending or copyright-free audio
   - Manual step: Browse and select audio track

6. **Design Assets**
   - Tool: Canva Free
   - Action: Create carousels, thumbnails
   - Manual step: Design in Canva and export

## Manual Instagram Tasks

These require direct interaction with Instagram:

1. **Upload Post**
   - Action: Post content to Instagram
   - Manual step: Use Instagram app/web to upload with generated caption, hashtags, alt text

2. **Engage with Audience**
   - Action: Reply to comments, DMs
   - Manual step: Spend 20 minutes engaging with followers

3. **Track Metrics**
   - Tool: Instagram Insights / Metricool Free
   - Action: Monitor saves, comments, shares
   - Manual step: Check analytics and note performance

4. **Share to Story**
   - Action: Share post to Instagram Story
   - Manual step: Use Instagram app to share post

5. **Schedule Posts**
   - Tool: Metricool Free
   - Action: Schedule future posts
   - Manual step: Use scheduling tool to plan content

## Weekly Manual Tasks

1. **Analyze Top-Performing Posts**
   - Review engagement metrics
   - Identify patterns in successful content

2. **Research Trending Reels Audio**
   - Browse Instagram Reels
   - Note trending audio tracks

3. **Collaborate with Creators**
   - Reach out to small creators
   - Plan collaborations

4. **Update Hashtags List**
   - Research new relevant hashtags
   - Update hashtag strategy

5. **Refresh Avatar Outfit/Voice Tone**
   - Update avatar appearance
   - Adjust voice settings if needed

## Notes

- The app automates: Idea generation, script creation, caption generation, hashtag suggestions, alt text generation
- All video/audio creation and editing must be done manually using external tools
- Instagram posting and engagement must be done manually
- Use the checklist in the app to track which automated tasks are complete and which need manual completion
